# Sanctum-402: Anonymous AI Gateway

Sanctum-402 is a decentralized, privacy-focused AI Gateway that allows users and autonomous agents to access uncensored LLMs (via Venice.ai) using cryptocurrency micropayments on the Base Network.

**No KYC. No Accounts. Just Wallet signatures.**
